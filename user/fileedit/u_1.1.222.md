## 概要

ここでは、ユーザーページで、DirectCloudに保存されているファイルのアクセス履歴を確認する方法について説明します。

---

## 制限・補足事項

本記事には、共有フォルダの操作方法について記載していますが、「マイボックス」「共有」「文書管理」「Connect」「DLP」「ウォームストレージ」でも、同様の手順で操作することができます。  

コールドストレージでは、アクセス履歴を確認できません。  
ファイルをコールドストレージから共有フォルダに復元すると、コールドストレージでの操作をアクセス履歴で確認できます。  

「副管理者」「副管理者1」「副管理者2」、および「オーナー」のアクセスレベルが付与されているユーザーがアクセス履歴を確認することができます。  

ゲストは、アクセス履歴を確認できません。  

Connect Userは、アクセスレベルが付与されている自社のConnectフォルダのアクセス履歴は確認できますが、招待されたConnectフォルダのアクセス履歴は確認できません。  

「アクセス履歴」画面には、直近6ヶ月のアクセス履歴が最大50件まで表示されます。  

アクセス履歴は、日時の降順で表示されます。アクセス履歴を並べ替えて表示することはできません。  

以下の操作はファイルに対するアクセスの対象とならないため、アクセス履歴には記録されません。  

- フォルダに対する操作  
- 文書属性に関する操作  
- 「コピーのキャンセル」  
- 「リンク閲覧」  
- 「リンクダウンロード」  
- 「復元リクエスト」  

契約プランがエンタープライズで、DirectCloudの無料オプションである「DirectCloud-CONNECT」を利用できる場合、「Connect」メニューが表示されます。  

契約プランがビジネス以上で、DirectCloudの有料オプションである「DirectCloud-SHIELD DLP」を契約している場合、「DLP」メニューが表示されます。  

契約プランがビジネス以上で、DirectCloudの無料オプションである「DirectCloud 電子帳簿保存法対応」を利用できる場合、「文書管理」メニューが表示されます。  

契約プランが「ビジネス」以上の場合、標準でウォームストレージを利用できるため、ユーザーページに「ウォームストレージ」メニューが表示されます。  

契約プランが「ビジネス」以上で、有料オプションの「コールドストレージ」を契約している場合、「コールドストレージ」メニューが表示されます。

## 手順

### アクセス履歴を確認する

1. アクセス履歴を確認したいファイルが保存されているフォルダをクリックします。  

   以下のいずれかの方法で、ファイルのアクセス履歴を表示します。  

   ❶ ファイルにチェックを入れて、上部にある「アクセス履歴」ボタンをクリックします。  
   ❷ ファイルを右クリックして表示されたメニューの「アクセス履歴」をクリックします。  
   ❸ ファイルの右側にある▼ボタンをクリックして、表示されたメニューの「アクセス履歴」をクリックします。  

   「アクセス履歴」画面に、ファイルのアクセス履歴が表示されます。  

| No. | 項目 | 説明 |
| --- | --- | --- |
| ❶ | ファイル名 | 現在表示されているアクセス履歴の対象ファイル名が表示されます。 |
| ❷ | アクセス履歴 | 操作：アクセス履歴の対象となった操作が表示されます。<br>詳細については、アクセス履歴の対象となる操作を参照してください。<br><br>ユーザー：ファイルを操作したユーザー名が表示されます。<br>ゲストとConnect Userには、ユーザー名の先頭にバッジ●が表示されます。<br>自動移動や自動削除のアクセス履歴には、「SYSTEM」と表示されます。<br>ユーザーにカーソルをあわせると、ツールチップでユーザーIDを確認できます。<br>「操作」が「アップロード」で、アップロード専用メールアドレス、リンク、受取フォルダへアップロードされた場合、メールアドレスが表示されます。<br>左に表示されている「i」アイコンにカーソルをあわせると、メールアドレスの情報を確認できます。<br><br>日時：アクセス履歴の対象の操作が行われた日時が表示されます。 |

---

### アクセス履歴の対象となる操作

ユーザーがファイルに対して以下の操作をすると、ファイルへのアクセス履歴が記録されます。  

これらの操作は、管理ページの「ログ」＞「ファイルの操作」メニューにも記録されます。  

| 操作 | 説明 |
| --- | --- |
| アップロード | マイボックス、共有フォルダ、文書管理フォルダ、Connectフォルダ、DLPフォルダ、ウォームストレージ、コールドストレージのフォルダへファイルをアップロード<br>ユーザーAPIの「ファイルのアップロード」を実行 |
| 上書き保存 | マイボックス、共有フォルダ、Connectフォルダ、DLPフォルダ、ウォームストレージのフォルダにファイルをアップロードしたあとに、上書き保存<br>ファイルの内容を変更してオンライン編集画面を終了 |
| 名前の変更 | マイボックス、共有フォルダ、文書管理フォルダ、Connectフォルダ、DLPフォルダ、ウォームストレージ、コールドストレージのフォルダで、フォルダまたはファイルの名前を変更<br>ユーザーAPIの「フォルダ名の変更」「ファイル名の変更」を実行 |
| 移動 | マイボックス、共有フォルダ、文書管理フォルダ、Connectフォルダ、DLPフォルダ、ウォームストレージのフォルダで、フォルダまたはファイルを移動 |
| コピー | マイボックス、共有フォルダ、文書管理フォルダ、Connectフォルダ、DLPフォルダ、ウォームストレージのフォルダで、フォルダまたはファイルをコピー |
| コールドストレージから復元 | コールドストレージから共有フォルダにファイルを復元 |
| 以前のバージョンに復元 | マイボックス、共有フォルダ、Connectフォルダ、DLPフォルダのファイルを、以前のバージョンに復元 |
| ゴミ箱 | 各フォルダでフォルダまたはファイルを削除（ゴミ箱に移動） |
| ゴミ箱から復元 | ゴミ箱からフォルダまたはファイルを復元 |
| 削除 | ゴミ箱を使用しない設定時にフォルダまたはファイルを削除 |
| 自動削除（ファイル期限） | ファイル期限機能による自動削除 |
| 自動削除（フォルダプロパティ） | フォルダプロパティ機能による自動削除 |
| オンライン編集 | オンライン編集画面を表示 |
| プレビュー | ファイルをプレビュー |
| DirectCloudドライブで開く | DirectCloud ドライブでファイルを直接閲覧 |
| オフラインに保存 | DirectCloud ドライブのオフラインフォルダに保存 |
| ダウンロード | ファイルをダウンロード |
| 添付ファイル送信 | 添付ファイル送信機能で送信 |
| リンク作成 | フォルダまたはファイルのリンクを作成 |
