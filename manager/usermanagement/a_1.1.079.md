## 概要

ここでは、「ユーザー管理」メニューで、指定したグループにユーザーを新規で追加する方法について説明します。

---

## 制限・補足事項

管理者権限が付与されたユーザーは、管理者の機能制限で、「ユーザー」にチェックが入っている必要があります。  

ユーザー一覧は、登録日の降順で表示されます。並べ替えることはできません。  

同じメールアドレスを使用して、複数のユーザーIDを作成できます。  

ユーザーIDは、一度設定すると、管理ページの画面上では変更できません。  
変更する場合は、[ユーザーIDを変更する方法]()を参照してください。  

契約担当管理者（一般管理者に契約担当管理者の権限を委譲する方法で契約担当管理者の権限を付与されたユーザー）および一般管理者（ユーザーに一般管理者の権限を追加する方法で一般管理者に追加されたユーザー）を削除することはできません。そのため、契約担当管理者および一般管理者の「操作」列に削除ボタンは表示されません。  

ユーザー情報を修正しても反映されない場合は、Webブラウザのキャッシュを削除したあとに、再度修正してください。  

ユーザーが削除された場合、そのユーザーのユーザーIDを再利用できるようになります。ただし、エンタープライズプランで監査オプションが適用されている場合、削除済のユーザーIDを再利用することはできません。  

ユーザーIDの複数人での利用は、以下のセキュリティリスクがあるため、推奨していません。  
　万が一、不正ログインがあった場合に、判断することができない  
　実際にログインしている人が誰なのかがわからない  
　実際に操作した人が誰なのかがわからない  

「ユーザー管理」メニューのグループは親子関係を持つツリー構造で表示されていますが、ユーザーを追加する際はそれぞれ独立したグループとして扱う必要があります。  

たとえば、ユーザーを追加したグループに下位階層のグループがある場合、下位階層のグループにユーザーは自動で追加されません。  

それぞれのグループには、個別にユーザーを追加してください。

---

## 手順

1. 「ユーザー」＞「ユーザー管理」メニューを選択し、ユーザーを追加したいグループをクリックして、上部にある「ユーザー追加」ボタンをクリックします。  

   「ユーザー追加」画面が表示されます。

2. ユーザー情報を入力して、「登録」ボタンをクリックします。  

| 項目 | 内容 |
| --- | --- |
| ユーザーID | ログインする際にユーザーの識別に使用されるIDを、3～250文字以内で設定します。<br>使用できる文字：半角英数字、記号（- _ @ . + #） |
| 名前 | ユーザーの表示名を、250文字以内で設定します。<br>使用できる文字：UTF-8の漢字、ひらがな、英大小文字、数字、スペース、多言語文字、記号および絵文字など |
| パスワード | 「パスワード登録用メールを送信」にチェックを入れて「登録」ボタンをクリックすると、パスワード登録用のメールがユーザーに送信されます。<br><br>NOTE：<br>パスワード登録用メールの有効期限は、送信後24時間以内です。<br>受信したパスワード登録用メールからパスワードを設定するユーザーの操作については、[パスワード登録用メールからパスワードを設定する方法]()を参照してください。<br><br>パスワードに使用できる文字や有効期限などのパスワード定義は、以下の手順で設定できます。<br>[ログインパスワードの文字種を制限する方法]()<br>[ログインパスワードの有効期間を設定する方法]()<br>[ユーザー・ゲストに過去のパスワードを再利用させない方法]()<br>[ログインパスワードの連続入力を制限する方法]()<br>[ログインパスワードの最小文字数を変更する方法]()<br>[ユーザー・ゲストがログインする際に強制的にパスワードを変更させるように設定する方法]() |
| メールアドレス | ユーザーのメールアドレスを、250文字以内で設定します。<br>使用できる文字：半角英数字、記号（- _ @ . + #） |
| 携帯電話 | 携帯電話番号を、50文字以内で設定します。<br>二要素認証を介してログインする際に、ワンタイムパスワードを送付する宛先として使用されます。<br>二要素認証を介したログインに成功すると、隣に「認証済み」と表示されます。<br>使用できる文字：半角数字記号（+ -） |
| 言語 | 以下の言語を設定します。<br>ユーザーページの表示言語<br>ユーザーに送付される通知メール<br><br>ユーザーページの表示言語を変更する方法の手順で、ユーザー自身で設定することもできます。 |
| GMT変更 | ユーザーのタイムゾーンを設定します。 |
| 割当容量 | マイボックスの最大容量を、契約プランの総ストレージ容量内で設定します。<br>共有フォルダ内の容量は含まれません。<br>使用できる文字：1以上の半角数字（単位：MB）<br>1024MBは1GBとして換算されます。 |
| 有効期限 | ユーザーに有効期限を設定する場合は、チェックを入れて、カレンダーで日付を設定します。 |
| 所属グループ | ユーザー追加先のグループを指定します。<br>「ユーザー追加」ボタンをクリックしたときに選択されていたグループが、デフォルトとして登録されます。最上位のグループ(会社名)は、自動的に所属グループに追加されます。 |
| 機能制限の設定 | ユーザーの機能制限を設定します。<br>グループの設定値に依存する<br>ユーザー追加先のグループに適用されている「機能制限の設定」が反映されます。<br><br>このユーザー専用の権限を設定する<br>「機能制限の設定」に表示されている設定名を選択して、使用可能な機能を適用できます。<br>「機能制限の設定へ」ボタンをクリックすると、「セキュリティ」＞「機能制限の設定」メニューに移動します。 |
| アクセス制御 | ユーザーに、「アプリケーション」「デバイス管理」「二要素認証」「IPアドレス」によるアクセス制御を適用します。<br>ここでは、「基本設定」または「アクセス制御の個別設定」で追加した個別設定を選択します。<br>「設定」ボタンをクリックすると、「セキュリティ」＞「アクセス制御の個別設定」メニューに移動します。<br><br>以下の手順で、すべてのユーザーに適用された設定情報が、デフォルトとして反映されます。<br>[ユーザーが利用できるアプリケーションを制限する方法]()<br>[ユーザーが利用できるデバイスを制限する方法]()<br>[ユーザーにワンタイムパスワードを用いたログインを強制する方法]()<br>[ユーザーページへログイン可能なIPアドレスを制限する方法]() |
