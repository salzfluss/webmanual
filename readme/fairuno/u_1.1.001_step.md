## 概要

ここでは、Webアプリケーションで、DirectCloudに保存されているフォルダ・ファイルへのリンクを作成・送付するための「リンク作成」画面を表示し、画面構成と基本的な機能を確認する方法について説明します。

リンクを送付するまでの基本的な手順については、【Webアプリ・ドライブ】共有リンクを作成して送付する方法［手順］を参照してください。

***

## 制限・補足事項

動作条件や制限・補足事項を含む、仕様に関する情報については、【Webアプリ】リンクの作成画面を表示して基本的な機能を確認する方法［仕様］を参照してください。

## 手順

### リンクの作成画面を表示する

{% stepper %}
{% step %}
### フォルダまたはファイルを選択

「共有」メニューをクリックし、共有リンクを作成したいフォルダまたはファイルが保存されている第1階層以降のフォルダをクリックします。
{% endstep %}

{% step %}
### 「リンクの作成」画面を表示する

以下のいずれかの方法で、「リンクの作成」画面を表示します。

* リンクを作成したいフォルダまたはファイルにチェックを入れて、左上にある「リンクの作成」ボタンをクリック
* リンクを作成したいファイルを右クリックして、「リンクの作成」をクリック
* リンクを作成したいファイルの右側にある▼ボタンをクリックして、表示されたメニューの「リンクの作成」をクリック

「リンクの作成」画面が表示されます。
{% endstep %}
{% endstepper %}

***

### 「リンクの作成」画面の基本構成

| No. | 設定項目               | 説明                                                                                                         |
| --- | ------------------ | ---------------------------------------------------------------------------------------------------------- |
| ❶   | ファイル・フォルダ名         | リンク作成の対象となるフォルダまたはファイルの名前が表示されます。                                                                          |
| ❷   | リンク                | 生成されたリンクURLが、「https://link.directcloud.jp/○○○○」という形式で表示されます。                                               |
| ❸   | 簡易設定／詳細設定の切り替えスイッチ | <p>初期状態では、詳細設定の画面が表示されています。<br>「簡易設定」リンクをクリックすると、簡易設定の画面に切り替わります。再度、「詳細設定」リンクをクリックすると、詳細設定の画面に切り替わります。</p> |
| ❹   | 簡易設定の設定項目          | リンクを作成する際の基本的な設定を行うエリアです。                                                                                  |
| ❺   | 詳細設定の設定項目          | <p>リンクを作成する際に詳細な設定を行うエリアです。<br>アドレス帳もこのエリアに表示されます。</p>                                                     |
| ❻   | 「プレビュー」ボタン         | クリックすると、プレビュー画面で、送信されるメールの内容を確認できます。                                                                       |
| ❼   | 「送信」ボタン            | 設定したリンクをメールで送信します。                                                                                         |
| ❽   | 「キャンセル」ボタン         | 設定をキャンセルして、「リンクの作成」画面を閉じます。                                                                                |
